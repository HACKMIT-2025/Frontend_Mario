🎮 Mario Game 通关功能测试

测试说明

这个页面用于测试Mario游戏的通关弹窗和排行榜功能。

Backend URL: https://25hackmit--hackmit25-backend.modal.run

🎯 测试步骤：

  • 1. 点击"启动测试游戏"加载游戏
  • 2. 使用方向键控制Mario移动
  • 3. 到达终点触发通关
  • 4. 查看通关弹窗是否正常显示
  • 5. 测试输入昵称并提交成绩
  • 6. 查看排行榜是否正常显示

🎮 游戏控制：

  • ← → 方向键：左右移动
  • ↑ 方向键或空格：跳跃
  • R键：重启游戏
  • P键：暂停/继续
